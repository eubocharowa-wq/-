# ЦБИ — сайт (репозиторий)

Статический сайт **Центра Бизнес-Инвестиций**: приветственная страница и полноценная основная версия с калькулятором и демо-скринингом.

## Структура репозитория

| Путь | Назначение |
|------|------------|
| `index.html` | Приветственный лендинг (точка входа) |
| `welcome.css`, `welcome.js` | Стили и логика приветственной страницы |
| `src/index.html` | Основной сайт: разделы, скрининг, калькулятор |
| `src/styles.css`, `src/app.js` | Стили и скрипты основного сайта |
| `src/assets/` | Логотип `logo.png`, папка `objects/` для PDF по запросу |
| `cursor-context/` | Контекст для позиционирования и текстов ЦБИ |
| `notes/`, `references/` | Заметки и исходные материалы проекта |

## Быстрый старт

Требуется Node.js (для локального сервера через `npx`). В корне репозитория:

```bash
npm start
```

Откройте в браузере указанный адрес (по умолчанию порт **5173**). Стартовая страница — корневой `index.html`, далее переход на `src/index.html`.

Без Node.js можно открыть `index.html` через «Открыть файл» — ссылки на `src/index.html` и ассеты работают относительно корня папки.

## Приветственная страница

- Кратко объясняет формат сайта и ведёт на основную версию.
- Опция **«Не показывать при следующем визите»** сохраняется в `localStorage` и при следующем заходе на корень выполняется переход сразу на основной сайт (ключ `cbi_skip_welcome`).

## Основной сайт

Экспертное позиционирование ЦБИ: проверка, цифры, сопровождение; ипотечный калькулятор; демо-скрининг (данные не отправляются на сервер). Подробнее см. файлы в `cursor-context/`.

### Ассеты

- Логотип подключается как `src/assets/logo.png` (при необходимости замените файл на фирменный, сохранив имя или обновите пути в `index.html` и `src/index.html`).
- PDF для блока «Материалы» положите в `src/assets/objects/` (см. `src/assets/objects/README.md`).

## Публикация

Подойдёт любой хостинг статики (GitHub Pages, Netlify, Vercel, S3): корень сайта = корень репозитория, главная страница — `index.html`.

Если сайт будет в подпапке (например `https://user.github.io/repo/`), поправьте относительные пути в `index.html`, `welcome.js` и в ссылке «Старт» в `src/index.html` под ваш `base`.
